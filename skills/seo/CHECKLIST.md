# SEO Checklist

## Metadata

Every public page must have:

- Unique title
- Unique description
- Canonical URL
- Robots directive
- Open Graph metadata
- Twitter Card metadata
- Language declaration
- Viewport
- Theme color (optional)

---

## Titles

- Unique
- 40–60 characters
- Human readable
- Descriptive

Recommended format

```
Page · Product
```

---

## Descriptions

- Unique
- 120–160 characters
- Natural language
- No keyword stuffing

---

## Canonical

Every public page must include:

```
<link rel="canonical">
```

Canonical URLs must:

- Use HTTPS
- Exclude query parameters
- Use one hostname
- Be absolute URLs

---

## Robots

Public pages

```
index,follow
```

Private pages

```
noindex,nofollow
```

---

## Sitemap

Must include:

- Public pages
- Blog
- Documentation
- Product pages

Must exclude:

- Login
- Register
- Dashboard
- Settings
- Admin
- Internal tools
- Drafts
- Preview pages

---

## Open Graph

Every public page should include:

- og:title
- og:description
- og:type
- og:image
- og:url

---

## Twitter

Use

```
summary_large_image
```

---

## Accessibility

Every page should include:

- One H1
- Descriptive headings
- Semantic HTML
- Image alt text

---

## Performance

Prefer:

- Server-side metadata
- Static generation where appropriate
- Optimized images
- Fast loading
