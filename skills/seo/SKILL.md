# SEO Standard Skill

## Purpose

Provide a consistent, production-ready SEO standard for modern web applications.

This skill enables AI agents to design, audit, maintain, and improve SEO implementations following Google Search best practices while preventing common indexing, discoverability, and duplicate content issues.

---

## Goals

- Improve search engine discoverability
- Maximize indexing of public content
- Prevent duplicate content
- Standardize metadata
- Generate reusable SEO configurations
- Improve maintainability
- Reduce manual SEO work
- Follow modern web standards

---

## Scope

Applicable to:

- SaaS applications
- Marketing websites
- Documentation sites
- Blogs
- Landing pages
- E-commerce
- Developer portals
- Open source projects
- Multi-tenant applications

Framework independent.

Supports modern frameworks including React, TanStack Start, Next.js, Astro, Remix, Nuxt, SvelteKit, Hono, Express, Laravel, Rails, Django and custom SSR solutions.

---

## Responsibilities

When invoked, always:

- Audit existing SEO implementation
- Validate page metadata
- Verify canonical URLs
- Review robots directives
- Validate sitemap generation
- Review robots.txt
- Validate Open Graph metadata
- Validate Twitter metadata
- Validate Structured Data
- Detect duplicate content
- Detect indexing problems
- Recommend improvements
- Generate reusable implementations
- Produce an audit report

---

## Principles

- Prefer reusable solutions over duplication.
- Metadata should be generated automatically whenever possible.
- Public pages should be indexable.
- Private pages should not be indexed.
- Every page should define a canonical URL.
- Follow Google's Search Essentials.
- Keep implementations framework agnostic.
- Never sacrifice accessibility for SEO.
